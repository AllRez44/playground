# NestJS v10 - TypeScript Starter (Docker)
## Instructions
### How to Install
 - `docker build -t nestjs . `
### How to Run
- `docker run -p 3000:3000 --name nestjs -v ./:/usr/src/app nestjs-starter`
---
### Docker Info
- **Image name:** nestjs-starter
- **Container name:** nestjs
- **Container Port:** 3000
